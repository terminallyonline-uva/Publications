# Peripherals Issue 1 (2025)

Welcome to the archive directory for **Issue 1** of Peripherals. This folder contains the individual article derivations and files for the issue.

Peripherals (Short: PAL, ISSN: 3117-5090) is a peer-reviewed, student-led, open-access journal in Media Studies. Peripherals is a project of Terminally Online, the masters' student association in Media Studies at the University of Amsterdam. We use GitHub as our file repository. 

You can find a complete catalogue of our publication on our website: [terminallyonline.nl/peripherals](https://terminallyonline.nl/peripherals).
