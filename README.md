# Peripherals (PAL) Archive

Peripherals (Short: PAL, ISSN: 3117-5090) is a peer-reviewed, student-led, open-access journal in Media Studies. Peripherals is a project of Terminally Online, the masters' student association in Media Studies at the University of Amsterdam. We use this GitHub repository as our primary file archive for published issues and articles.

You can find a complete catalogue of our publications on our website: [terminallyonline.nl/peripherals](https://terminallyonline.nl/peripherals).
